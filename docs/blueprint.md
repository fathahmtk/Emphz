# **App Name**: Emphz

## Core Features:

- Dynamic Product Catalog: Display GRP products dynamically with photos, names, specifications, and quick view options.
- Project Case Studies: Showcase project case studies with before-after images, project details, location, and client type.
- Admin CMS Dashboard: Enable the admin to manage product images, specifications, and project case studies. This feature uses a generative AI tool to assist content creation and management within the CMS.
- Contact Form with Lead Tracking: A contact form that uses Firebase Functions for email and lead tracking, powered by a generative AI tool that will reason about when to incorporate contact information into marketing campaigns.
- Technical Downloads: Offer downloadable technical brochures and product data sheets (PDF links).
- Integrated SEO: Implement SEO schema, metadata, and sitemap for better search engine visibility.
- Social and Location Integration: Integrate social media links, WhatsApp link, and a Google Maps embed for office and factory locations.

## Style Guidelines:

- Primary color: Deep Navy Blue (#003049) for a strong and reliable impression.
- Secondary color: Light Beige (#EAE2B7) to provide a soft contrast and warmth.
- Accent color: Burnt Orange (#D62828) to highlight key information and calls to action.
- Body and headline font: 'Roboto' for a modern, clean, and highly readable design.
- Use minimalist icons in white and light beige to represent product categories and solutions.
- Minimalist UI with a focus on glassmorphism and subtle 3D design elements for product cards.
- Implement animated scroll transitions and subtle motion effects to enhance user experience.
